### 为什么归档？

归档旧文档的原因：

1. **消除重复** - 多个文档内容重叠，造成维护困难
2. **提升清晰度** - 统一的文档结构更易于导航
3. **保持更新** - 单一文档源更容易保持同步
4. **保留历史** - 归档而非删除，保留历史参考价值
